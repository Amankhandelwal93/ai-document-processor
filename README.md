# AI Document Processor

Extracts structured data from PDFs using LangChain + OpenAI GPT.

## What it does
- Reads PDF documents (invoices, forms, reports)
- Extracts targeted fields using LLM
- Outputs clean JSON → exports to Excel

## Tech Stack
- Python
- LangChain
- pdfplumber
- OpenAI API

## How to Run
pip install -r requirements.txt
python main.py
